## üöÄ Getting Started

### Quick Launch (MyBinder)
Try our interactive notebooks without installation:

[![Launch Notebooks](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/DawsonInstitute/hts-coils/main?urlpath=lab/tree/notebooks/)

```bash

# Install dependencies

python scripts/validate_reproducibility.py
```
